Analyzing survey data with R

How to tabulate in R frequencies of a survey answers (weighted)

When exploring survey data I often like to have a look at the frequencies of answers to particular questions, broken down by respondent’s country, sex, age or any other variable at hand. Howver, this task ain’t that simple, as survey data are often coded in very inconvenient way, for instance:

id weight gender age question_1 question_2 …
1 1.120 male 20 agree agree NA
2 0.980 male 26 very agree very disagree NA
3 1.022 female 34 agree agree NA
4 1.071 female 21 neigher agree, nor disagree agree NA
5 0.943 male 54 disagree agree NA

With other types of data you would normally render table of frequencies with sort of the following code:

data %>% 
  count(question_1) %>% 
  mutate(prop = prop.table(n))

However, the characteristic feature of surveys is that each observation is assigned a unique weight (so that the sample of respondents reflects the structure of relevant population). One needs to multiply each variable by relevant weights to obtain correct averages, frequencies and other statisics. But how would you do that with character variable? Solution seems simple - create dummy variables for each level of character variable (“strongly”, “agree”, etc.) and multiply them by weights. However, coding this in R is a bit tedious.

Fortunately, there is an easier solution, namely “weights” package, which contains a function wpct, which was designed to do exactly that. Let’s have a look on how does it work.

# Let's save the list of answers to variable, for later use. 
answers <- names(wpct(data$Question, weight = data$Weight))

# We need to apply wpct to each answer and each group separately. Fortunately wpct works with margritr pips, so we don't need to resolve to loops. 

freq_table <- data %>% 
  group_by(Gender) %>%
  do(as.data.frame(wpct(.$Question, .$Weight, na.rm = TRUE))) %>%
  mutate(answers = answers)

# Now let's do few finishing touches: change ugly default name of second column and spread the table to make comparisons easier. 

names(freq_table)[2] <- "frequency"
freq_table <- freq_table %>% 
  spread(key = answers, "frequency")

# And voila...
freq_table
